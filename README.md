# דשבורד עסקי

אפליקציית ניהול לידים, אירועים ומשימות לעסק.

## תכונות

- **דשבורד** — KPIs, גרפים, השוואה שנתית וחודשית
- **לידים** — הזנה/עריכה/מחיקה, סינון, חיפוש
- **אירועים** — כל ליד שמסומן "נסגר" עובר אוטומטית לדף האירועים
- **ניתוח המרות** — מקורות, סוגי אירועים, סיבות אי-סגירה, פאנל המרה ותובנות
- **משימות** — תזכורות אוטומטיות

## תזכורות אוטומטיות

- **ליד "פולואפ"** — תזכורת כל יומיים (5 תזכורות)
- **ליד "נסגר" (אירוע)** — שלוש תזכורות אוטומטיות:
  1. שבוע לפני האירוע — שליחת שאלון לחתן
  2. יום לפני האירוע — וידוא מילוי שאלונים
  3. יום אחרי האירוע — שליחת קבלה

## טכנולוגיה

- React 18 + Babel Standalone
- Chart.js
- אחסון מקומי (localStorage) — הנתונים נשמרים בדפדפן

## פריסה

`.github/workflows/deploy.yml`.
